GROQ API CONFIGURATION

Get a free key at console.groq.com/keys
Your key is stored only on your device and never sent anywhere except Groq's API.

How it works:
1. Go to console.groq.com/keys
2. Click Create API Key
3. Paste it above and click Save
4. Click the Wiredge icon in your toolbar to start compressing!
