Privacy Policy — Wiredge

Last updated: May 2025

What data we collect

Wiredge does not collect, store, or transmit any personal data to our servers. The extension runs entirely in your browser.

API Key

Your Groq API key is stored locally using Chrome's chrome.storage.local API. It never leaves your device except to make direct requests to api.groq.com on your behalf.

UI Descriptions

The text you type into Wiredge is sent directly to Groq's API for DSL compression. We do not log, store, or see this data. Please refer to Groq's Privacy Policy for how they handle API requests.

No tracking

Wiredge contains no analytics, no tracking pixels, no cookies, and no third-party scripts.

Permissions

storage — to save your API key locally.
clipboardWrite — to let you copy the generated DSL prompt with one click.
