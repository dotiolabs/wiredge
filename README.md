# Wiredge ⚡

**Wiredge** is the ultimate AI Prompt Compressor and Context Manager. Built for power users who demand efficiency, it allows you to compress prompts to save API tokens and seamlessly switch your active conversation between Claude, ChatGPT, Gemini, and Grok with a single click.

> Designed and built by **Vaibhav Singh** & **dotiolabs**

## Repository Structure

This repository is split into two primary components:

1. **`/extension`** - The core Chrome Extension source code.
2. **`/website`** - The custom landing page and promotional website for the extension.
