# Wiredge Landing Page

This is the promotional website and landing page for the Wiredge Chrome Extension.

It features a custom "Dark Glassmorphism" aesthetic designed to perfectly mirror the UI of the extension itself.

## Tech Stack
To ensure maximum performance and zero load time, this site is built entirely without heavy frontend frameworks:
- **HTML5:** Semantic structure.
- **Vanilla CSS3:** Custom styling utilizing CSS variables, Flexbox/Grid, and `backdrop-filter` for premium glass effects.
- **Vanilla JavaScript:** Lightweight scroll intersection observers and micro-animations.

## How to View Locally
You do not need a local server to view this website. Simply double-click on `index.html` to open it directly in your web browser.

## How to Deploy
Because this is a static site, you can host it for free on almost any platform in seconds:

- **Netlify:** Drag and drop this folder into [Netlify Drop](https://app.netlify.com/drop).
- **Vercel:** Import this repository into Vercel and set the Root Directory to `/website`.
- **GitHub Pages:** Enable Pages in your repository settings and set the source folder.
